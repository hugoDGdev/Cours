## Logique formelle et algèbre de Boole

#### Forme Basique :
Suite de déductions simple (restaurant) : 
- Dessert?
	- if Profiteroles? 
			=> True
		- else if desserts != maison 
			  => False
			- else if accompagné || dessert au chocolat 
			  => True

#### Formalisation mathématique :
On associe un symbole formel à chaque proposition :
	D : Prend dessert
	M : Fait maison
	A : Accompagné
	C : |Dessert choco| >= 1
	P : Profiteroles
==> D vrai si (M vrai et A vrai) ou (M vrai et C vrai) ou P vrai
===> D = (M  · A) + (M · C) + C
